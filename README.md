### File merger

## this is a high performance,lightweight Command-Line Interface (CLI) python application that merges your chosen pdf files into a pdf file called 'Combined_Document.pdf'.

## how to use

append your chosen pdf files into the 'fileMerger' folder then run the 'merger.py' file by typing 'python merger.py' in the terminal for windows.

## Features

* **[Easy use]:** [Organizes files easily by running the application.]
* **[portable]:** [Zero Dependencies, Built using standard libraries making it portable.]
* **[not limited]:** [can merge many pdf files quickly.]
